# Package Management

This article is about:
- Managing packages in Hatch

You will learn about:
- Adding, listing, and removing packages

See the [Quick Start Guide](../users/quick_start.md) for basic package operations.

## Add a Package

```bash
hatch package add <package-name>
```

## List Packages

```bash
hatch package list
```

## Remove a Package

```bash
hatch package remove <package-name>
```

For advanced usage, see [Creating Packages](creating_packages.md).
