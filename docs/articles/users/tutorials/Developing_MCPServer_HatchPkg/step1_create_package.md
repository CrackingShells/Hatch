This article is about:
- Creating a new MCP server package with Hatch

You will learn about:
- Initializing a package
- Understanding the basic structure

This is the first step to develop your own MCP server using Hatch. The example uses the `arithmetic_pkg`.
This tutorial assumes that Hatch has been installed following 

## Step 1: Create a New Package

Run the following command in your terminal from any other directory:

```bash
hatch create arithmetic_pkg --description "A simple arithmetic MCP server"
```

This will generate a new package directory with the required files.
