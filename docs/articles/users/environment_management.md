# Environment Management

This article is about:
- Advanced environment operations in Hatch

You will learn about:
- Managing, configuring, and troubleshooting environments

See the [Quick Start Guide](../users/quick_start.md) for basic usage.

## Listing Environments

```bash
hatch env list
```

## Setting the Current Environment

```bash
hatch env use <env-name>
```

## Removing an Environment

```bash
hatch env remove <env-name>
```

For troubleshooting, see the [Error Handling Guide](error_handling.md).
