# Glossary

This article is about:
- Definitions of terms used in Hatch documentation

You will learn about:
- The meaning of key terms and concepts

This glossary provides definitions for terms and concepts referenced throughout the documentation. Use it as a reference for unfamiliar terminology.

## Terms

- **MCP**: Model Context Protocol
- **Environment**: An isolated context for packages and dependencies
- **Package**: A distributable unit containing code and metadata
- **Registry**: A service for storing and retrieving packages
- **Conda**: A package and environment manager
- **Miniforge**: A minimal installer for Conda
- **PATH**: System environment variable for executable search
- **PlantUML**: Tool for creating diagrams in documentation

For more foundational concepts and step-by-step guides, see the appendices.
